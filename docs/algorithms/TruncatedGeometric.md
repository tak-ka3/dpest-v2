# TruncatedGeometric

## アルゴリズムの説明

TruncatedGeometricは、切断幾何分布に基づくノイズ機構です。幾何分布（離散版のラプラス分布）を有限範囲に切断して使用します。

**プライバシー保証**: $0.12$ -差分プライバシー

## モード

**サンプリングモード**

## プライバシー損失結果

| 項目 | 値 |
|------|-----|
| 入力サイズ | 5 |
| 推定 ε | 0.1312 |
| 理論 ε | 0.12 |
| 誤差 | +0.0112 (+9.3%) |
| 実行時間 | 23.92秒 |

**データソース**: `docs/privacy_loss_report.md`

## 理論的な計算量

**サンプリングモード**: $O(N \times m) = O(5 \times 10^6)$ 演算

## 理論的な誤差（精度）

**モンテカルロ誤差**: $O(1/\sqrt{N}) \approx 10^{-3}$ ## 理論と実験結果の比較分析

### 精度と速度

| 項目 | 値 | 評価 |
|------|-----|------|
| 相対誤差 | 9.3% | やや大きい |
| 実行時間 | 23.92秒 | サンプリングモードとしては標準的 |

### 比較: DP-Sniper vs StatDP vs DPEST

| 手法 | 推定 ε | 実行時間 |
|------|--------|----------|
| DP-Sniper | 0.116 | 10秒 |
| StatDP | 0.117 | 120秒 |
| DPEST | 0.1312 | 23.92秒 |

**結論**: DPESTは理論値に最も近い（誤差9.3%）。実行時間はDP-Sniperより2.4倍遅いが、StatDPより5倍高速。
